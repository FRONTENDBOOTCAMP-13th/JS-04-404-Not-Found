## 프로젝트 소개

### 프로젝트 목적

- JavaScript 학습 합니다.
- 추억의 포켓몬 게임을 경험 합니다.
- Gacha 와 Slot 미니 게임을 통해, 포켓몬 카드를 수집합니다.
- 수집한 포켓몬을 도감에서 확인하고, 그 외 포켓몬 타입별 분류하여 포켓몬 정보를 수집합니다.
